# Sort

You can sort folders and files in different ways. The selected order is independent for each main area, such as the Library, [Master folders](master-folders), [Servers](servers), and the file navigation view while browsing.

OpenComic also keeps browsing order and reading order separate, so changing how items are listed in one context does not necessarily change how pages are read inside a comic.

---

### Sort options in Library, Master folders, Servers, etc

- **Name:** Sort items alphabetically by name.
- **Name / Number:** Ensures that numbers in names are sorted by their numeric value instead of alphabetically.
- **Number:** Sort purely by numeric value extracted from item names.
- **Last add:** Sort by the most recently added items.
- **Last reading:** Sort by the most recently read items.
- **Modified date:** Sort by the file or folder modification date.
- **Shuffle:** Randomize the order of the items.
- **Invert:** Invert the sort.

---

### Sort options in navigation

- **Name:** Sort items alphabetically by name.
- **Name / Number:** Ensures that numbers in names are sorted by their numeric value instead of alphabetically.
- **Number:** Sort purely by numeric value extracted from item names.
- **Folders first:** Shows folders before images. **Note:** it doesn’t currently differentiate between folders and compressed files.
- **Compressed files first:** Lists compressed files before regular files in the current navigation view.
- **Invert:** Invert the sort.
