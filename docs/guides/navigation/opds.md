# OPDS

OpenComic supports most OPDS features.

----

## Features

- OPDS `1.x` and `2.0` support.
- OpenComic `Continue reading` and `Recentrly added`.
- Search support.
- Pagination support.
- Authentication support (Basic and Digest)
- Show all publication data.
