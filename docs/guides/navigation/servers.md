# Servers

OpenComic can connect to different local or external servers using various protocols.

----

### Supported protocols

- `smb://`
- `ftp://`
- `ftps://`
- `scp://`
- `sftp://`
- `ssh://`
- `s3://`
- `webdav://`
- `webdavs://`
